# get-etymology
Asynchronous Wiktionary etymology scraper for Russian words.

## Example usage
```
база базис фундамент
*Этимология слова база.* Происходит от лат. basis «основание, подножие», далее из др.-греч. βάσις «ход, шаг; основание», от гл.  βαίνω «ходить», из праиндоевр. *gwem-. Русск. база встречается со времени Петра I, заимств. через нем. Base или франц. base из лат. Использованы данные словаря М. Фасмера. См. Список литературы.

*Этимология слова фундамент.* Происходит от лат. fundamentum «фундамент, основание», далее из fundāre «снабжать основанием или дном; основывать», далее из fundus «дно, основание», далее из праиндоевр. *bhu(n)d- «дно».

*Этимология слова базис.* Происходит от лат. basis «основание, подножие», далее из др.-греч. βάσις «ход, шаг; основание», от гл.  βαίνω «ходить», из праиндоевр. *gwem-. Русск. база встречается со времени Петра I, заимств. через нем. Base или франц. base из лат. Использованы данные словаря М. Фасмера. См. Список литературы.
```
